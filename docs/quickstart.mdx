---
title: Quickstart
description: Track your first agent action in 5 minutes.
---

## 1. Create an API key

Sign in at [app.quorvel.tech](https://app.quorvel.tech), then
**Settings -> API keys -> Create key**. Copy the secret (shown once).

## 2. Install the SDK

```bash
npm install @quorvel/sdk
```

## 3. Track an action

```ts
import { Quorvel } from "@quorvel/sdk"

const qrv = new Quorvel({ apiKey: process.env.QUORVEL_API_KEY })

await qrv.track({
  idempotencyKey: "order-123",
  tool: "refund.issue",
  args: { amount: 5000 },
})
```

## 4. See it in the dashboard

Open the dashboard - your action appears in the activity list, and any action
that requires approval shows up in the Approvals queue.
